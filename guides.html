Downloading Web Application

Here are the steps required to install the web application on your device
This will allow the site to be opened offline as an application with the only missing feature is the question paper pdfs / periodic table pdf, otherwise the website is fully functional in offline mode

Platforms:

  1. 1-Windows
  2. 2-iOS
  3. 3-Android

Windows

1-Click on download icon at top right

2-Confirm install in alert

iOS

1-Click on share icon

2-Select add to Home Screen option from the bottom

3-Confirm add button in menu

4-Application added to Home Screen

Android

1-Click on download icon at top right

2-Select install app option from dropdown menu

3-Confirm install in alert

4-Confirm add button in widget menu

5-Application added to Home Screen
